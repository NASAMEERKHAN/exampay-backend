🚀 PaySphere– Secure Exam Fee & Hall Ticket Management System

Developed a secure backend system for exam fee management using Spring Boot. 
Implemented JWT-based authentication and role-based authorization for ADMIN and STUDENT roles. 
Integrated BCrypt for password security and deployed MySQL database on Railway cloud. 
Designed REST APIs for secure payment handling and user management.

🔗 Live Website: https://payshpere.netlify.app/

⚙️ Backend API: https://exampay-backend.onrender.com

💡 About the Project

PaySphere is a real-world full-stack web application designed to manage student fee payments, verification, and hall ticket generation.

📌 This project was built to solve a real problem —
My college doesn’t have a proper digital system for handling fee payments and approvals.
So I developed a complete end-to-end solution from scratch.

🔄 System Workflow
👨‍🎓 Student Module
Secure login using unique ID 🔐
View fee details & dashboard
Make payment 💳
Instant receipt generation 📄

🧑‍💼 Admin Module
View all student records
Track payment status (Paid / Pending)
Notify students if payment is pending 🔔
Forward verified payments

🏫 HOD Module
Receive verification requests
Approves student status
Issue hall ticket 🎫
